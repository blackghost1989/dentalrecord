# 🦷 獸醫牙科記錄表

互動式獸醫牙科檢查紀錄應用程式，支援犬貓牙齒檢查與治療計畫記錄。

## ✨ 功能特色

- **互動式牙科圖表** — 直接點擊解剖學正確的犬/貓牙科圖表上的牙齒進行記錄
- **逐齒評估** — 每顆牙齒可記錄詳細臨床發現：
  - 缺牙（Missing）
  - 動搖度指數（Mobility Index M0–M3）
  - 分岔病變（Furcation F1–F3，僅顯示於多根牙）
  - 牙齦退縮（Gingival Recession，mm）
  - 牙周囊袋深度（Periodontal Pocket Depth，mm）
  - 齒槽骨流失（Alveolar Bone Loss：<25%、25–50%、>50%）
  - 骨折類型（簡單/複雜 牙冠/牙根骨折）
  - 牙齦炎指數（Gingivitis Index 0–3）
  - 牙結石指數（Calculus Index 0–3）
  - X 光發現（自由文字）
- **治療計畫** — 每顆牙齒可標記建議治療：
  - 牙周治療 / 根管治療 / 修復 / 拔牙 / 翻瓣手術
  - 自動排斥邏輯（選擇拔牙時自動停用牙周/根管/修復）
- **自動牙周病分級**
  - **PD1** — 牙齦炎指數 1–3（僅牙齦炎，無骨流失）
  - **PD2** — 骨流失 <25% 或分岔 F1
  - **PD3** — 骨流失 25–50% 或分岔 F2
  - **PD4** — 骨流失 >50% 或分岔 F3
- **檢查摘要** — 自動產生所有有臨床發現的牙齒摘要
- **PDF 匯出** — 一鍵匯出完整圖表與檢查結果為 PDF
- **JSON 備份/匯入** — 匯出與匯入 JSON 格式紀錄檔
- **本地歷史紀錄** — 儲存與讀取瀏覽器 localStorage 中的紀錄
- **犬貓支援** — 分別的牙科圖表（犬 42 齒、貓 30 齒）
- **響應式設計** — 桌面與行動裝置自適應佈局

## 🚀 快速開始

### 前置需求

- Node.js ≥ 18
- npm

### 安裝與執行

```bash
# 安裝相依套件
npm install

# 啟動開發伺服器
npm run dev
```

應用程式將在 `http://localhost:5173` 上運行。

### 建置

```bash
# 正式環境建置（Web）
npm run build

# 單一檔案建置（自包含 HTML）
npm run build:single

# Electron 桌面應用
npm run electron:dev        # 開發模式
npm run electron:build      # 打包（當前作業系統）
npm run electron:build:win  # 打包（Windows）
```

## 📖 使用方式

1. **填寫病患資料** — 飼主姓名、寵物名、物種、年齡等
2. **點擊牙齒** — 在牙科圖表上點擊目標牙齒，開啟評估面板
3. **記錄檢查結果** — 選擇動搖度、分岔、骨流失、牙齦炎指數等
4. **標記治療計畫** — 勾選建議的治療項目
5. **切換牙齒** — 使用 ← → 箭頭或直接點擊其他牙齒
6. **檢視摘要** — 向下捲動查看自動產生的檢查摘要與牙周病分級
7. **匯出** — 點擊「導出 PDF」下載 PDF，或「備份」匯出 JSON
